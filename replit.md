# INMU大富豪 (INMU Daifugou)

## Project Overview
A browser-based card game (大富豪 / Daifugou / "Great Millionaire") with INMU meme themes. Built with React + Vite frontend and a Node.js WebSocket backend server.

- **Frontend**: React 18 + TypeScript + Vite, runs on port 5000
- **Backend**: Node.js WebSocket server (`server/index.cjs`), runs on port 3001
- **Start command**: `npm run dev` (runs both via `start.sh`)

## Features
- Single-player vs CPU mode
- Online multiplayer via WebSocket rooms (friend mode, X recruit mode)
- Special card effects (Kuronuri, 7-pass, 10-discard, etc.)
- PWA support with service worker
- Audio/BGM system
- Player profiles and friend system
